# HRMS Portal

Copy your existing HRMS codebase into this folder.

## Setup

```bash
# From the root of this project:
cp -r /Users/madhu/Documents/Codex/hrms-portal/* ./hrms-portal/

# Install and run
cd hrms-portal
npm install
npm run dev
# Runs on http://localhost:3000
```

The Employee Login button on the marketing site will automatically
connect to http://localhost:3000/dashboard when running locally.
